Metax

Metadata Repository for Research Data

Metax is a metadata storage for Finnish research data. It is a part of Fairdata Services and acts as a metadata repository for all research data stored in Fairdata Services. In addition to using Qvain Tool to create datasets, Metax also offers an API (Metax REST API) to create and update the dataset metadata.
